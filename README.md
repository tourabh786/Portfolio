# Torabh Jatav Portfolio

Modern animated portfolio website built with pure HTML, CSS, and JavaScript.

## Overview
This is a multi-page personal portfolio focused on:
- Core Banking & Digital Banking expertise
- SQL engineering and automation work
- AI-assisted web development projects

## Features
- Dark premium UI theme
- Animated preloader (3D spinner)
- Custom cursor chase effect (desktop)
- Smooth reveal animations on scroll
- Responsive mobile navigation
- Contact form flow via WhatsApp
- Live project links on Home and Work pages

## Pages
- `index.html` - Home + highlights + live links
- `about.html` - Profile, experience, education
- `skills.html` - Technical and domain skills
- `projects.html` - Live deployments + banking projects
- `contact.html` - Contact details + WhatsApp form

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- Font Awesome
- Google Fonts

## Run Locally
Because this is a static site, you can run it in two ways:

1. Open `index.html` directly in your browser.
2. Or use a local server (recommended):
   - VS Code Live Server extension, or
   - any static server like `npx serve`.
